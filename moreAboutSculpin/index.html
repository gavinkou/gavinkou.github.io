关于Sculpin

本系统使用Sculpin构建，Sculpin以Symfony为框架开发，markdown的处理部份使用michelf/php-markdown，进行解析，php-markdown支持基本语法和扩展语法，

  • 请留意扩展语法不为基本语法支持，请谨慎使用，详细的中文使用说明见： MDExtSyn

  • 自定义css支持，这部分由用户自己提供，基于sass实现，目前自定义了这样几个css class，这些class的使用，只需要在Typora的代码块：

    • .formula
    • .herb
    • .quote
    • .emphasis
    • .comment
    ```{.formula}
处方： 桂枝 10g  芍药15g  生姜15g
```


    ```.comment
这是注释
```


    ```{.herb}
本草： 麻黄
```


    ```{.quote}
引用内容
```


    ```{.emphasis}
重点内容
```

  • 高亮Highlight支持

    利用MDExtSyn的代码块class或者#id支持功能，可以设置各种语言的高亮（由highlight.js 提供支持）

另外，还支持别的MD convert，比如说commonmark，使用： composer require league/commonmark 来安装。但是本系统使用michelf/php-markdown。
