西行漫记（02）之实际线路

2019-07-15 雲文 雲文

西行漫记 游记

工欲善其事必先利其器，简单罗列重要准备事项

  • 交通工具：SUV一辆，昂科威汽油四驱版

  • 修车工具，

    • 14的套筒与棘轮扳手是必备工具，换胎用。其余型号的套筒若干
    • 千斤顶
    • 加气泵及胎压计
    • 拖车绳及挂钩
    • 其余钳剪胶工具若干

  • 全尺寸备胎一枚，之所以全尺寸，与行车电脑搭在的胎压监控芯片不支持备胎，索性很多厂家默认的备胎是非全尺寸。即便是行车电脑不支持的原因，也不用担心非全尺寸备胎小一圈，在差速锁激活时，左右轮不一样，出现速度差，而强制偏向。

  • 手机两部：

    • 智能一部，一部日用，导航APP以及离线地图和导航数据，其余APP按需准备。
    • 超长待机一部：用以应急。

  • 现金若干

  • 油卡两张：中石油、中石化各一。西北四省的中石油、中石化，独立核算，注意圈存余额。

  • 水具，必备，冷热水各一升/人，每天发车前，检查，沙碛遍地，有水应急时，就不至于出现脱水等状况。
  • 应急干粮，饼类为佳，不易变质，路上随走随更新

  • 药品，针灸针和艾条在没有合适的药物时，可救急，在甘肃天水，真的就碰上了牙疼，还好针灸救急。

    • 中暑
    • 伤风感冒
    • 消化道药物
    • 针灸针
    • 艾条

  • 露营装备若干

    • 帐篷，西北风大，地钉尤其重要，地广人稀，野物出没，备而不用。
    • 睡袋，同上
    • 茶具，饮马大草原，举杯黄沙中，开车不喝酒，但是能喝茶

事后根据百度地图的数据显示：

整个行程线路，东西横跨四千五百公里，直线距离横跨4353公里，行驶里程两万余公里，历时45天。

由衷的感谢百度地图，

全程提供导航，数据分析，路线规划，甚至有时候连酒店都用百度地图搞定，价钱并不比携程有太大差异。

此处亦可作为全文的游记目录汇总。

西行段

  1. 上海 → 马鞍山 → 合肥 → 河南南阳
  2. 河南南阳 → 陕西西安
  3. 陕西西安 → 陕西终南山
  4. 陕西终南山 → 陕西扶风法门寺
  5. 陕西西安 → 甘肃天水
  6. 甘肃天水 → 甘肃兰州 → 青海西宁 → 青海湖
  7. 西宁青海湖 →青海祁连县扁都口 → 青海祁连县
  8. 青海祁连县 → 甘肃张掖
  9. 甘肃张掖 → 甘肃敦煌
  10. 甘肃敦煌 → 星星峡 → 哈密
  11. 新疆哈密 → 新疆吐鲁番
  12. 新疆吐鲁番 → 新疆乌鲁木齐
  13. 新疆乌鲁木齐 → 奎屯/独山子（独库公路起点）
  14. 独库公路 → 新疆库车
  15. 新疆库车 → 新疆阿克苏 → 新疆喀什

截此为止，到达行程的最西端，开始折返上海：

  1. 新疆喀什 → 新疆莎车 → 新疆叶城 → 新疆和田
  2. 新疆和田 → 新疆民丰
  3. 新疆民丰 → 新疆且末
  4. 新疆且末 → 新疆若羌
  5. 新疆若羌 → 新疆茫崖
  6. 新疆茫崖 → 青海德令哈
  7. 青海德令哈 → 西宁湟源县塔尔寺
  8. 塔尔寺 → 甘肃兰州 → 陕西乾县
  9. 陕西乾县 → 安徽亳州
  10. 安徽亳州 → 上海
