西行漫记（二）之实际线路

By 雲文 雲文 | 2021-05-10

西行漫记

整个行程线路，东西横跨四千五百公里，直线距离横跨4353公里，历时45天。

以最西端的喀什为界，分为往西和折返两段，

折返

此处亦可作为全文的游记目录汇总。

西行段

  1. 上海 → 马鞍山 → 合肥 → 河南南阳
  2. 河南南阳 → 陕西西安
  3. 陕西西安 → 陕西终南山
  4. 陕西终南山 → 陕西扶风法门寺
  5. 陕西西安 → 甘肃天水
  6. 甘肃天水 → 甘肃兰州 → 青海西宁 → 青海湖
  7. 西宁青海湖 →青海祁连县扁都口 → 青海祁连县
  8. 青海祁连县 → 甘肃张掖
  9. 甘肃张掖 → 甘肃敦煌
  10. 甘肃敦煌 → 星星峡 → 哈密
  11. 新疆哈密 → 新疆吐鲁番
  12. 新疆吐鲁番 → 新疆乌鲁木齐
  13. 新疆乌鲁木齐 → 奎屯/独山子（独库公路起点）
  14. 独库公路 → 新疆库车
  15. 新疆库车 → 新疆阿克苏 → 新疆喀什

截此为止，到达行程的最西端，开始折返上海：

  1. 新疆喀什 → 新疆莎车 → 新疆叶城 → 新疆和田
  2. 新疆和田 → 新疆民丰
  3. 新疆民丰 → 新疆且末
  4. 新疆且末 → 新疆若羌
  5. 新疆若羌 → 新疆茫崖
  6. 新疆茫崖 → 青海德令哈
  7. 青海德令哈 → 西宁湟源县塔尔寺
  8. 塔尔寺 → 甘肃兰州 → 陕西乾县
  9. 陕西乾县 → 安徽亳州
  10. 安徽亳州 → 上海
